=== API Response

==== Response Body

[cols="10%,10%,10%,70%"]
|===
| Path | Type | Mandatory | Description

| `ok`
| Boolean
| Y
| Success(true), Fail(false)

| `result`
| Object
| N
| Api Result

| `error`
| String
| N
| Error Code
|===

==== API Success Example

----
HTTP/1.1 200 OK
Content-Type: application/json

{
    "ok": true,
    "result": {
      "key": "value"
    }
}
----

==== API Fail Example

----
HTTP/1.1 500 Internal Server Error
Content-Type: application/json

{
  "ok": false,
  "error": "internal_error"
}
----
