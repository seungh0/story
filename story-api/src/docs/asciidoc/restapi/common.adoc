== Common Documentation

=== API Response

==== Response Body

[cols="10%,10%,10%,70%"]
|===
| Path | Type | Mandatory | Description

| `ok`
| Boolean
| Y
| Success(true), Fail(false)

| `result`
| Object
| N
| Api Result

| `error`
| String
| N
| Error Code
|===

==== API Success Example

----
HTTP/1.1 200 OK
Content-Type: application/json

{
  "ok": true,
  "result": {
    "key": "value"
  }
}
----

==== API Fail Example

----
HTTP/1.1 500 Internal Server Error
Content-Type: application/json

{
  "ok": false,
  "error": "internal_error"
}
----

----
HTTP/1.1 400 Bad Request
Content-Type: application/json

{
  "ok": false,
  "error": "invalid_arguments",
  "messages": [
    "[title] must not be blank",
    "[content] size must be between 0 and 500"
  ]
}
----
