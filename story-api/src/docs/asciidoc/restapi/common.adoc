== 공통 정보

=== API 응답 모델

==== Response Body

[cols="10%,10%,10%,70%"]
|===
| Path | Type | Mandatory | Description

| `ok`
| Boolean
| Y
| 성공(true), 실패(false)

| `result`
| Object
| N
| API 결과

| `error`
| String
| N
| 에러 코드 ("에러 코드 참고")

| `reasons`
| Array<String>
| N
| 에러 추가 설명 목록 (debug 용)
|===

==== API Success Example

----
HTTP/1.1 200 OK
Content-Type: application/json

{
  "ok": true,
  "result": {
    "key": "value"
  }
}
----

==== API Fail Example

----
HTTP/1.1 500 Internal Server Error
Content-Type: application/json

{
  "ok": false,
  "error": "internal_error"
}
----

----
HTTP/1.1 400 Bad Request
Content-Type: application/json

{
  "ok": false,
  "error": "invalid_arguments",
  "messages": [
    "[title] must not be blank",
    "[content] size must be between 0 and 500"
  ]
}
----

==== API Warning Example

----
HTTP/1.1 200 OK
Content-Type: application/json

{
  "ok": true,
  "result": {
    "key": "value"
  }
}
----
