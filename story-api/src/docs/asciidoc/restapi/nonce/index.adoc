== Nonce

=== About Nonce

- "Nonce"란, 중복 호출을 방지하는 데 사용되는 임의의 값으로, 멱등성이 없는 API에서 선택적으로 사용할 수 있습니다.

==== Nonce 사용 방법

- Nonce Create API에서 Nonce 값을 가져옵니다.
- 생성된 Nonce 토큰를 파라미터로 함께 사용할 API를 호출합니다.

==== Nonce 정책

- Nonce는 "expirationSeconds"로 설정한 기간 동안만 유효합니다.
- Nonce는 한 번 할당되고 한 번만 사용할 수 있습니다.

위를 만족하지 않는 경우 'invalid_nonce' 에러가 발생합니다.
